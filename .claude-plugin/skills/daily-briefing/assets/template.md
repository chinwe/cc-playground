# 每日简报模板

此模板定义了每日简报的输出格式。

---

## 简报结构

1. **标题**: 包含日期的简报标题
2. **操作日志**: 记录抓取过程的日志信息
3. **主题分类**: 按主题分组的内容
4. **新闻条目**: 每条包含标题、链接、简介

## 示例输出格式

```markdown
# 每日简报 - 2026-03-01

## 操作日志

- [08:00] 开始抓取内容...
- [08:01] 从3个RSS源获取12条内容
- [08:02] 从2个网页获取5条内容
- [08:03] 共获取17条内容
- [08:03] 简报生成完成

## 技术

### [新框架发布](https://example.com/article1)
某框架发布了2.0版本，带来了显著的性能提升和新特性。

### [AI研究突破](https://example.com/article2)
研究人员在机器学习领域取得重要进展，新算法提高了准确率。

## 新闻

### [行业动态](https://example.com/article3)
某公司发布新产品，引起市场广泛关注。

## 开发

### [最佳实践](https://example.com/article4)
分享代码审查的最佳实践和工具推荐。
```

## 自定义样式

如需自定义简报样式，可以修改 `scripts/generate_report.py` 中的 `generate_markdown_report` 函数。
