# daily-use

Collection of daily utility skills for productivity and information gathering.

## Skills

### daily-briefing

Generate comprehensive daily briefings to help users start their day organized and informed.

**Features:**
- Weather information (conditions, temperature, forecast)
- News summaries (top headlines from various sources)
- Calendar/agenda overview (meetings and events)
- Priority task list (top action items)

**Usage:**

Use this skill when you request:
- "daily briefing" or "morning summary"
- "what's on my agenda today?"
- "what should I focus on today?"
- Summary of news, weather, or upcoming events

**Output Format:**

The skill outputs a structured markdown briefing with:
- Header with date and greeting
- Weather section (if applicable)
- News highlights
- Today's agenda
- Top priorities

## Configuration

The daily-briefing skill can be configured with custom RSS sources, web scraping targets, and output templates. See `skills/daily-briefing/` for configuration files.
